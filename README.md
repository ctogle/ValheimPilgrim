# Pilgrim

Pilgrim introduces new mechanics that make it easier to avoid backtracking. Activate forsaken powers from any item stand, upgrade the cart for the nomad lifestyle, and make offerings to the flames for abilities and guidance.

## Requirements

- [BepInExPack Valheim](https://thunderstore.io/c/valheim/p/denikson/BepInExPack_Valheim/)
- [YamlDotNet](https://thunderstore.io/c/valheim/p/ValheimModding/YamlDotNet/)
